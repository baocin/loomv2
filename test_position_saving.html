Loom Pipeline Monitor - Position Saving Test

Test Instructions

  1. Click "Open Pipeline Monitor" to open the main app in a new tab
  2. In the new tab, drag any node to a different position
  3. Open browser dev tools (F12) and check the Console for position saving logs

  5. Come back to this tab and click "Test LocalStorage" to verify data was saved
